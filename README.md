Instruções de Execução e Compilação:

Projeto da Unity 2022.3 LTS

No caminho Assets/Scripts/Managers é possível encontrar um Scriptable Object com o nome PlatformSettings, que possui uma variável booleana que controla as diferenças entre mobile e PC, caso ativa, o projeto funcionará apenas como mobile, para testes em PC, apenas desativar no inspector.

Compilação:

O APK pode ser feito a build de maneira simples pela Unity:

File > Build Profile... > Build

ou com um cabo USB conectado em um celular:

File > Build and Run

Escolha onde salvar a build dentro do seu aparelho mobile e se divirta!
